## ğŸš€ Getting Started

1. Clone the repository:
```bash
git clone https://github.com/Namraa310806/WeatherAppReactJS.git
```

2. Install dependencies:
```bash
cd WeatherAppReactJS
npm install
```

3. Create a `.env` file in the root directory and add your API keys:
